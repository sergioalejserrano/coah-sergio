Modo recomendado. Los datos de Garmin se actualizan automaticamente cada 4h via GitHub Actions.
Con un token, el botón ↻ baja datos frescos de Garmin al instante (dispara el sync). Sin token, ↻ solo recarga el último sync automático (cada 4h). Crear token → fine-grained, repo coah-sergio, permiso Actions: Read & write.
¿Amigos quieren su propia app?
Setup para amigos (~10 min, gratis):
1. Cuenta en github.com
2. Fork del repo (1 clic)
3. Settings → Secrets → Agregar: GARMIN_EMAIL, GARMIN_PASSWORD, DATA_PASSPHRASE
4. Settings → Pages → branch main
5. Actions → Sync Garmin → Run workflow
6. Abrir usuario.github.io/coah-sergio
⚠ Garmin bloquea IPs de servidores. Este modo puede fallar con error 429. Se recomienda el modo Frase.
